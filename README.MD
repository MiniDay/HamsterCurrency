# HamsterCurrency
仓鼠的多货币插件  
目前只支持BC跨服模式  
**现已支持Vault经济系统！**

# 依赖前置
- [HamsterAPI](https://github.com/MiniDay/HamsterAPI/releases)
- （仅跨服模式需要）[HamsterService](https://github.com/MiniDay/HamsterService/releases)


# 开发者

## 依赖导入

### Maven
```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```
